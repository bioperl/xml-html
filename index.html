www.bioxml.org

bioxml.org is a resource to gather XML documentation, DTDs and tools for biology in one central location.

  • An page about XML hosted on bio.perl.org.

    www.bioxml.org hopes to expand from this start and take on a much more repository orientated role towards XML in biology. For example, keeping the DTDs in a open repository and providing tools in many languages which work with the biological XML formats proposed here.
